copyWithin()

Array.prototype.copyWithin(target, start = 0, end = this.length)

  • target(必选)：从该位置开始替换
  • start(可选)：从该位置开始读数据，默认为0，负数表示倒数
  • end(可选)：到该位置停止，默认为数组长度，负数为倒数
  • 三个数为数值，不是则自动转化为数值
