总结

  • 当 new 一个函数的时候会创建一个对象，被创建对象.__proto__ === 函数.prototype
  • 一切函数都是由 Function 这个函数创建的，被创建函数.__proto__ === Function.prototype
  • 一切函数的原型对象都是由 Object 这个函数创建的，Object.prototype === 一切函数.prototype.__proto__
