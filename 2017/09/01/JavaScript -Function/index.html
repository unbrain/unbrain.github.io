函数的本质

函数其实是一种特殊的 Object，可以反复调用该代码块

this 与 arguments

在调用函数时大家都会理所当然的使用，f() 类似这样的方式，但是我们还可以用f.call(this, arguments) 这样的方式

this 若为 undefined 一般的话是会被看成 window
